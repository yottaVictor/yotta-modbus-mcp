```

```

---

## ⚠️ 注意事項

- 測試新增設備時使用假設備（名稱「測試設備」），測完後移除避免弄髒 config
- 涉及實際設備（辦公室/測試機/一樓電箱）的寫入操作會真實影響硬體，請確認安全後再測試
- 所有工具的 `deviceId` 參數傳設備 **name**（例如「辦公室」），不是型號 id
